## Project Overview

This repository contains the **User Research & Feasibility Study** for the **QuickPay Fintech Mobile Wallet App**, completed as part of the UI/UX Internship. The project focuses on understanding user needs, identifying pain points in existing digital payment applications, and proposing a simplified, user-friendly mobile wallet experience.

---

## Project Objective

* Conduct user research for a fintech mobile application.
* Identify user goals, needs, and pain points.
* Create a user persona based on research findings.
* Analyze the user flow for peer-to-peer money transfers.
* Perform a competitive analysis of existing payment applications.
* Evaluate the feasibility of the proposed solution.
